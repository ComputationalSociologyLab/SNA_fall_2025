Centrality

  • Capture something about which nodes are most important to the network
  • Used as predictors of other outcomes, like attitudes, behaviors, etc.

Indegree Centrality

  • Measures how many ties each node receives
  • The number of people that talks to node i in a social way.
# 3 people talk to node 1, 1 person talks to node 2
indegree_social <- degree(graph = s641_social, mode = "in")

Closeness Centrality

  • Inverse of the mean distance in shortest path between a given node and all other nodes.
  • where:
  • \(n\) is the number of nodes in the network
  • \(d(v, u)\) is the geodesic distance (shortest path length) between node \(v\) and node \(u\)
    \[ C(v) = \frac{1}{\frac{1}{n-1} \sum_{u \neq v} d(v, u)} \]

Centralization

  • centr_degree() : Computes degree centrality statistics and network-level centralization
  • \(C_D(v_i)\) is the degree centrality of node \(i\)
  • \(C_D(v^*)\) is the maximum degree centrality observed in the network
    \[ C_D(G) = \frac{\sum_{i=1}^{n} \left( C_D(v^*) - C_D(v_i) \right)} {\max \sum_{i=1}^{n} \left( C_D(v^*) - C_D(v_i) \right)} \]

Centralization

  • Numerator : Difference btw max degree centrality and observed centrality for node \(i\)
  • Denominator :The maximum possible value of Numerator in a graph of size \(n\) (i.e., the star graph). \[ C_D(G) = \frac{\sum_{i=1}^{n} \left( C_D(v^*) - C_D(v_i) \right)} {\max \sum_{i=1}^{n} \left( C_D(v^*) - C_D(v_i) \right)} \]

Centralization

  • 0 ‚Üí All nodes have similar degrees (no centralization)
  • 1 ‚Üí One node has all the connections (perfectly centralized) \[ C_D(G) = \frac{\sum_{i=1}^{n} \left( C_D(v^*) - C_D(v_i) \right)} {\max \sum_{i=1}^{n} \left( C_D(v^*) - C_D(v_i) \right)} \]
